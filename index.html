EBB Exam Decoder

Instructor tool — Saint Petersburg College · Ethics Beyond Boundaries

Instructor use only. Paste the student's Submission Token from their Canvas submission below. The token contains a verification hash — if the readable report was modified after submission, the hash will not match and you will see an alert. This tool runs entirely in your browser; no data is sent anywhere.
✓ Verification passed — token matches content. Submission is authentic.
⚠ Verification failed — the token hash does not match. The report may have been modified after submission. Compare token content (shown below) with the submitted PDF.
